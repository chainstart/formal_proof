# WOWII induced-bipartite support package

This Lean package supports the combined manuscript on two Written on the Wall II
lower bounds for largest induced bipartite subgraphs.

It contains:

- `WowiiConjecture13`: a diameter plus local-neighborhood independence bound.
- `WowiiConjecture17`: an independence number plus one-third diameter bound.

Build the package with:

```bash
lake build
```

The root import is `AmraLibrary`.
